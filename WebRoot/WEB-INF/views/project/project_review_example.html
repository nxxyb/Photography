费用说明

1、机票：含全国往返昆明机票及机场税；

2、住宿：全程入住指定五星级酒店标准间；

3、用餐：6早9正，十人一桌，十菜一汤，餐标35元/人，特色餐50元/人（丽江火塘鸡、楚雄野生菌、大理砂锅鱼、石林彝王宴）

4、用车：正规营运旅游空调车，保证10%空座率；

5、门票：洱海游船、南诏风情岛、双廊、崇圣寺三塔、大理古城、丽江古城、玉龙雪山、冰川公园大索道、白水河、蓝月谷、三道茶白族歌舞表演、七彩云南、石林、彝家天天火把节晚会、花之城；

6、导游：怡之系列专职专线导游全程精致服务；

7、管家：专业客服和专职管家全程24小时为您服务；

8、保险：旅行社责任保险、旅游意外险、云南旅游组合险；

9、贴心：行程路途中爱游独家休息服务区，为您提供贴心服务

10、购物：丽江螺璇藻、大理白族民居银器店、昆明七彩云南、花之城（自愿购买，绝无强迫）

行程描述

  • 行程

  • 昆明—687km—磨憨

    早餐后出发，途经玉溪、峨山、元江、墨江、普洱、思茅、勐仑、勐远，途中穿越美丽的丛林高速公路一思小高速公路，是穿越原始森林的高速公路，它是昆明至曼谷国际大通道的重要路段，热带植物茂盛，景色优美，体现出了人与自然和谐发展，是一条真正的“车在林中行，人在画中游”的高速公路，傍晚抵达中老边境的磨憨小镇。
